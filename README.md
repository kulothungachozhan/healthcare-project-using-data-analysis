# healthcare-project-using-data-analysis
This project performs a basic statistical and visual analysis of a heart disease dataset using **NumPy**, **Pandas**, and **Matplotlib**.


 Project Objectives

- Analyze key health indicators (age, cholesterol, max heart rate)
- Identify high-risk patients based on cholesterol levels
- Measure correlation between age and cholesterol
- Visualize heart disease trends using plots

---

 Dataset Description

The dataset `heart.csv` contains patient medical records with attributes like:

- `age`: Patient age
- `chol`: Serum cholesterol in mg/dl
- `thalach`: Maximum heart rate achieved
- `target`: Heart disease diagnosis (1 = disease, 0 = no disease)
